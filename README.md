# React Age Calculator

This is a simple Age Calculator built with React.js and Luxon.

## Features

- User-friendly UI
- Date picker to select birthdate
- Calculates exact age in years, months, and days using Luxon

## Technologies Used

- React.js
- Luxon
- Vite (for project scaffolding)
